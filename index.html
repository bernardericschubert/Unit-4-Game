Emoji Game

Here is how this works:

You have 4 emojis to choose from below. Each of these carries a random point value that will not be shown to you. When you click it, the value will be added to the current score.

Your goal for this game is to choose emojis until you hit the target score. If you go over the total value, you lose.

Each time the game is reset (via win or loss), the value of the emojis will change. Good luck!
